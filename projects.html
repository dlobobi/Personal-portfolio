Household Management System (Andersons' Family)

Olpas Sensor Configuration & Calibration Tools

Category: Professional (Olpas) Lead Developer

Context & Background

Olpas required offline-capable, browser-based applications to allow field technicians to configure and calibrate industrial sensors via USB. Because the hardware relies on the Modbus RTU protocol (limited to 16-bit integers), there was a critical need for a mathematical bridge between human-readable decimals and rigid machine code to prevent hardware crashes.

Realisations & Role

As the lead developer, I overhauled the architecture and core mathematical engines. I transformed the UI into a dynamic JavaScript engine driven by a centralized SENSOR_REGISTRY. I also engineered a custom algorithm (`encodeModbusFloat`) that dynamically translates massive integers and floating-point decimals into 4-register Modbus Scientific Notation. Additionally, I built an offline simulator using `localStorage` for safe testing and authored the technical documentation.

Skills Gained

  • ▹ Hard Skills: JavaScript, Offline SPA Architecture, Web Serial API, Modbus RTU Memory Management, Linear Regression Algorithms.
  • ▹ Soft Skills: UI/UX Empathy for field technicians, autonomous edge-case debugging, translating complex math into maintainable documentation.

IoT Dashboards & Gateway Infrastructure

Category: Professional (Olpas) Systems Administrator

Context & Background

Olpas required a secure, high-performing data ingestion platform to monitor their IoT sensor fleets. The objective was to maintain a reliable telemetry pipeline while actively optimizing the existing server architecture to handle long-term data storage efficiently, ensure high system availability, and protect against external threats without interrupting live sensor data.

Realisations & Role

Taking over an existing containerized backend stack (Docker, Mosquitto, Node-RED, InfluxDB, Grafana), I served as the Systems Administrator managing critical infrastructure. My key realizations included:

  • ▹ Security & Networking: Hardened the Linux server using standard firewall utilities (UFW) and bot defenses (Fail2Ban). Resolved internal Docker routing conflicts to bypass public SSL traps and diagnosed UDP packet drops to ensure data integrity.
  • ▹ Database Optimization: Eliminated severe Grafana latency by cleaning up InfluxDB, engineering a strict 6-month data retention policy, and implementing automated database exports.
  • ▹ System Health & Reliability: Conducted deep-dive investigations into server resource allocation to diagnose abnormal data consumption and high usage spikes, optimizing backend performance to prevent bottlenecks during peak telemetry ingestion.

Skills Gained

  • ▹ Hard Skills: Linux Server Resource Management, InfluxDB Lifecycle/Retention Policies, Security (UFW, Fail2Ban), Docker Subnets, MQTT/Node-RED pipelines.
  • ▹ Soft Skills: Proactive Resource Optimization, Lifecycle Management, Technical Writing for internal Wikis, Scientific-method Troubleshooting.
